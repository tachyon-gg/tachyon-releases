# Tachyon

**Protect your game saves. Play anywhere.**

[Tachyon](https://tachyon.gg) is a cross-platform sync engine for serious gamers, modders, and streamers. It automatically backs up your game saves to the cloud in real-time as you play, lets you sync across devices, and provides timeline-based version control for your save files.

## Features

- **Auto-Detection** - Automatically finds and protects save files for thousands of games out of the box
- **Real-Time Cloud Backup** - Saves sync to the cloud as you play, with no manual intervention required
- **Timelines** - Restore any previous save or branch into alternate playthroughs with full version history
- **Cross-Device Sync** - Seamlessly switch between devices without losing progress
- **Zero Performance Impact** - Lightweight background process that won't affect your gaming experience

## Installation

1. Download the `.msi` installer from the [releases page](https://github.com/tachyon-gg/tachyon-releases/releases)
2. Run the installer and follow the on-screen instructions
3. Launch Tachyon from the Start Menu

## Getting Started

1. **Install and Scan** - Tachyon automatically detects your installed games
2. **Play** - Your saves are backed up in real-time as you play
3. **Restore or Sync** - Use the timeline viewer to restore previous saves or sync across devices

## Who It's For

- **Gamers** - Protect against hardware failures and switch devices without losing progress
- **Modders** - Experiment safely with mods and instantly roll back if something breaks
- **Content Creators** - Branch timelines for alternate storylines and access previous save states for recording
